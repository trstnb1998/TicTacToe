# Project Zero

## How To Play

#### Game Rules
* There will be 2 players and is a turn based game
* Player 1 is X (Crosses) and Player 2 is O (Naughts)
* Once either player connects 3 tiles. They win!
* If all tiles have been selected then game will be a draw
